# airtable2lowdb

Converts Airtable base to LowDB compatible JSON files

## Usage

### Clone

```bash
git clone https://github.com/applefreak/airtable2lowdb
cd airtable2lowdb/
```

### Configure

```bash
cp config.json.example config.json
vim config.json
```

Remember to change out the values!

### Install & Run

```bash
npm install
npm start
```

### Examine

```bash
cat output/*.json
```
